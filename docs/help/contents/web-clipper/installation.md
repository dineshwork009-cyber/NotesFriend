# Installation

## Chromium-based browsers

[Go to chrome webstore](https://chrome.google.com/webstore/detail/notesfriend-web-clipper/kljhpemdlcnjohmfmkogahelkcidieaj) to get the latest version of [web clipper for Notesfriend](https://notesfriend.com/notesfriend-web-clipper).

## Firefox-based browsers

1. Download the .xpi file from [here](https://notesfriend.com/notesfriend-web-clipper)
2. Open `about:addons`
3. Drag & drop the .xpi file
